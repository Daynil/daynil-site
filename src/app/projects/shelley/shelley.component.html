Shelley Pier Counseling

What is it?

A website created for a client with a personal counseling business. The client requested a website that showcases all of her subspecialties, as well as a way for potential clients to contact her right from the website.

Methodology

I worked with the client to create the website design and presentation. The client liked the snappy response times of a single page application style website, so I created the website using front end routing.

Tech Stack

  • Front end: Angular 2+
  • Back end: Node and Express
  • Hosting: Digital Ocean
